# BTP_2
## Photocatalytic Oxidation of Heterocyclic Compounds.
• Designed and developed a visible light-driven benzylic oxidation route for the synthesis of phenanthridine deriva-
tives using Na2 eosin Y as a photocatalyst.<br>
• Optimized reaction parameters including solvent, light source and reaction time, achieving an 74% yield.<br>
• Characterized synthesized compounds using TLC, 1H NMR, 13C NMR and IR spectroscopy.
